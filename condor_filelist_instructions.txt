* this file contains some instructions and examples of how to use condor_filelist.perl

* * * Synopsis: perl script that will submit a number of jobs to condor and
* * * * delegates a space to write your files based on the name of the python
* * * * configuration file supplied to the function.

* * * Usage: from the command line the form goes as follows

condor_filelist.perl --batch [number of jobs per batch] --prodspace [full path
to product space] ./myNuplizer/python/my_configuration_file_cfg.py
./list_of_files.txt
